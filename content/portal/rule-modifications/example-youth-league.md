# Example Youth League Rule Modifications

This document outlines the rule modifications for the Example Youth League.

## General Information

- **Age Groups**: U10, U12, U14
- **Game Format**: Modified periods appropriate for age group
- **Equipment**: Age-appropriate equipment as specified by league

## Rule Modifications

### Timing

| Rule | Modification |
|------|-------------|
| Period Length | Modified for age group |
| Halftime | 5 minutes |
| Timeouts | 2 per half |
| Overtime | Not applicable (ties allowed) |

### Game Play

| Rule | Modification |
|------|-------------|
| Defensive Restrictions | Age-appropriate restrictions apply |
| Playing Area | Modified as needed for age group |
| Substitutions | Free substitutions encouraged |
| Timing Rules | Running time for younger age groups |

### Penalties

| Rule | Modification |
|------|-------------|
| Personal Fouls | Standard limits apply |
| Team Penalties | Modified bonus rules |
| Technical Violations | Educational approach preferred |

## Notes for Officials

- Focus on teaching and development
- Encourage good sportsmanship
- Keep the game flowing
- Communicate clearly with coaches and players

## Contact

For questions about these modifications, contact the league coordinator.
